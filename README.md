Periphery Station 14

Periphery Station 14 is designed as an SCP experience within Space Station 14, it seems like a perfect setting to use through the SS14 format, even if a few modifications will need to made.

Periphery Station is forked from: Currently Nothing!

The choice to fork from this server was due to: It not currently existing in a large enough capacity!
